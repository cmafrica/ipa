This page is designed to be a convenient tool for linguists and linguistics students.
I first noticed the serious lack of an organized, central resource for the entire IPA in
Introduction to Linguistics, but it became much more apparent during Phonetics & Phonemics.
The main goal of this project was to use data mining to amass and organize a comprehensive
IPA database; all of the information exists online already, but it's scattered and not easily
combined. While this is only somewhat accomplished here so far, visualization of this information
is much more useful to others. I hope that you find this tool as useful as I would have found
it in my previous classes.

Please email me if you have any ideas, or if you come across an error or bug in the program.
This is still in Beta testing, and I am relying on your input to help make this tool as comprehensive,
useful and user-friendly as possible.

  • Downloadable PDF & XML Files Containing All Relevant Information
  • Conversion to Pure Javascript to allow for Downloading and Offline Usage
  • Games! A Flashcard Game, A Multiple Choice Game, A Game Testing
    Recognition of/Differentiation Between Sounds
